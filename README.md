LinearRoad_ActiveMQ
==========
This is a ActiveMQ implementation of LinearRoad Benchmark(http://www.cs.brandeis.edu/~linearroad/) which is completely written in Java.

##1. Introduction

This is an implementation of LinearRoad benchmark for Apache ActiveMQ.
